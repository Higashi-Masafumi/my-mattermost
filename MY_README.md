# アプリの始め方

1. [mattermost-desktopのリポジトリをcloneする](https://github.com/Higashi-Masafumi/mattermost)

2. アプリを次のコマンドでbuidする
```bash
$ npm install
$ npm run build
$ npm start
```

3. 立ち上がった画面で、先ほどの手順で得たサーバーのURLを入力する（ドキュメント通りに進めた場合は以下のURLになるはず）
```bash
http://localhost:8065
```
